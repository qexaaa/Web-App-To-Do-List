# Web App To-Do List

Aplikasi web **To-Do List** sederhana berbasis **PHP Native** dan **MySQL** yang digunakan untuk mencatat, mengelola, dan menyelesaikan tugas harian.

Aplikasi ini dibuat sebagai latihan pengembangan web menggunakan PHP tanpa framework, sehingga seluruh proses seperti koneksi database dan pengolahan data dilakukan secara manual.
